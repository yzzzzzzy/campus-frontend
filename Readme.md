# 🎓 大学生信息共享平台 (CampusInfo Share) - 项目规划文档

## 1. 项目概述
本项目旨在打造一个高价值的校园综合门户平台。本平台**立足于消除大学生之间的“信息差”**，打破校园内的资源壁垒。通过整合升学资讯、竞赛组队、职业发展及技能提升等核心板块，为在校大学生提供一站式、可检索的成长资源库，助力每一位学生更好地规划未来。

## 2. 技术栈架构
- **前端 (Frontend):** Vue 3 (Composition API) + Element Plus + Vue Router + Pinia
- **后端 (Backend):** Node.js + Express.js + JSON Web Token (JWT)
- **数据库 (Database):** MySQL 8.0+
- **工具链:** VS Code, Git, Postman, Navicat Premium Lite

## 3. 核心功能模块划分 (全量功能版)

### 3.1 📚 升学考公模块 (Study & Exam)
- **资讯速递:** 汇总考研院校招录信息、国家及地方公务员考试公告。
- **资料共享:** 提供历年考研真题、四六级资料、专业课笔记的下载与在线预览。
- **上岸经验:** 学长学姐考研/考公的备考路径与时间规划分享。

### 3.2 🚀 个人提升与技能进阶 (Personal Growth & Skills) - [NEW]
- **编程开发:** 汇总主流编程语言（Java, Python, C++）及前端框架（如 Vue.js, React）的学习路径与项目实战资源。
- **创意设计:** 针对自媒体时代，提供视频剪辑（PR/剪映）、摄影修图（PS/LR）及审美提升的系统教程。
- **办公效率:** 包含 PPT 高级制作、Excel 自动化以及时间管理等大学生必备职场软技能。

### 3.3 🏆 竞赛与大创模块 (Competitions & Projects)
- **赛事日历:** 汇总数学建模、蓝桥杯、互联网+、挑战杯等主流赛事的报名时间与要求。
- **组队招募:** 利用用户的专业 (`major`) 和技能 (`skills`) 标签，实现跨专业的高效组队。

### 3.4 💼 实习与就业模块 (Career & Internship)
- **内推广场:** 发布优质企业校招信息、大厂实习岗位及校友专属内推码。
- **面经系统:** 按岗位（研发、产品、设计等）分类的真实面试经验与真题回忆。

### 3.5 🤝 校园互助与生活模块 (Campus Life)
- **二手交易/失物招领:** 规范化的校园闲置流转与信息寻人。
- **互动问答:** 针对选课、校园政策、生活常识的互助问答区。

### 3.6 👤 个人空间与管理 (User & Admin)
- **档案中心:** 绑定专业、技能栈，记录个人发布、收藏与成长足迹。
- **后台控制:** 基于角色的管理系统，维护平台内容的健康与真实性。
