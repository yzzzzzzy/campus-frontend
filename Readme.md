# 🎓 大学生信息共享平台 (CampusInfo Share) - 项目全生命周期文档

## 1. 项目概述
本项目旨在打造一个高价值的校园综合门户平台。本平台**立足于消除大学生之间的“信息差”**，打破校园内的资源壁垒。通过整合升学资讯、竞赛组队、职业发展及技能提升等核心板块，为在校大学生提供一站式、可检索的成长资源库。

## 2. 技术栈架构
- **前端 (Frontend):** Vue 3 (Composition API) + Element Plus + Vue Router + Pinia
- **后端 (Backend):** Node.js + Express.js + MySQL 8.0 + JWT (身份验证)
- **版本控制:** Git (代码托管于 GitHub)
- **工具链:** VS Code, Postman, Navicat Premium Lite

## 3. 核心功能模块 (已完成框架搭建)

### 3.1 📚 升学考公 (Study)
- **形态:** 列表布局。
- **功能:** 提供考研、考公、四六级等分类资料的下载与链接。

### 3.2 🚀 个人提升 (Skills)
- **形态:** 响应式网格卡片布局。
- **功能:** 汇总编程、剪辑、办公效率等技能的学习资源。

### 3.3 🏆 竞赛组队 (Competition)
- **形态:** 瀑布流卡片。
- **功能:** 突出显示“招募中/已满员”状态及“急需技能”标签，支持查看联系方式。

### 3.4 💼 实习就业 (Career)
- **形态:** 招聘风格列表。
- **功能:** 发布校招内推、实习岗位及面试经验分享。

### 3.5 💬 校园论坛 (Forum)
- **形态:** 独立社区页面。
- **功能:** 支持发帖、分类检索、展开式评论互动。

### 3.6 👤 个人中心 (Profile)
- **形态:** 聚合面板。
- **功能:** 我的发布记录、获赞统计、个人资料修改（昵称/头像/密码）、全局退出登录。

## 4. 系统架构演进记录 (Architecture Evolution)

- **V1.0:** 基础论坛 demo，实现登录注册与简单发帖。
- **V2.0:** 引入主页“金刚区”导航，确立“消除信息差”的门户化方向。
- **V3.0 (重大重构):** - **组件化:** 抽离出全局通用的 `NavBar.vue`，实现 B 站风格的头像悬浮菜单。
    - **解耦化:** 将“校园论坛”从主页独立，实现“场景化发布”逻辑（每个模块独立发布）。
- **V4.0:** 接入 Git 版本管理，开始进行功能细节闭环（如个人资料设置）。

## 🚀 快速启动指南 

### 1. 数据库准备
1. 确保本地已安装 MySQL 8.0+。
2. 在 Navicat 中新建数据库 `campus_info_db`。
3. 导入本目录下的 `database/` 文件夹中的所有 SQL 文件（或使用总览 SQL）。
4. 修改后端的 `.env` 文件，将 `DB_PASSWORD` 改为您的本地数据库密码。

### 2. 一键运行项目


在目录 `Graduation project` 下运行：
`npm run start:all`
